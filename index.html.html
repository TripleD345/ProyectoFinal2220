Sobre el Proyecto

Proposito del Proyecto

Este sitio web representa nuestro recorrido como estudiantes de tecnología y futuros profesionales. Nuestro objetivo es mostrar nuestras habilidades, certificaciones, trabajo en equipo y pasión por la tecnología mediante el desarrollo web moderno.

Audiencia Deseada

Este sitio web está diseñado para instructores, compañeros y futuros empleadores interesados en estudiantes con ambición, creatividad y pasión por la tecnología.

Tecnologías y Habilidades

HTML5

CSS3

Diseños responsivos utilizando Flexbox, Grid y técnicas modernas de estilizado.
